# Contributing to Walk With Ibrahim

First off, thank you for considering contributing to this project. This is a humanitarian effort — every improvement directly helps Ibrahim get the medical care he needs.

## How Can I Contribute?

### Reporting Bugs

Found something broken? Please [open an issue](https://github.com/KJfamily33/ibrahim/issues/new) with:
- A clear, descriptive title
- Steps to reproduce the bug
- Expected vs actual behavior
- Screenshots if applicable
- Your browser/device info

### Suggesting Features

Have an idea to improve the platform? [Open an issue](https://github.com/KJfamily33/ibrahim/issues/new) with:
- A clear description of the feature
- Why it would help the campaign
- Any implementation ideas you have

### Code Contributions

1. **Fork** the repository
2. **Clone** your fork locally
3. **Create a branch** for your feature (`git checkout -b feature/your-feature`)
4. **Make your changes**
5. **Test** locally with `pnpm dev`
6. **Build** to check for errors (`pnpm build`)
7. **Commit** with a clear message
8. **Push** to your fork
9. **Open a Pull Request**

### Translation Help

The site supports Arabic and English. If you can help:
- Improve existing Arabic translations
- Add new languages (Turkish, Urdu, French, etc.)
- Fix any cultural/religious inaccuracies

Look for the `content` objects in components — they contain bilingual strings.

### Design Improvements

- UI/UX enhancements
- Accessibility improvements
- Mobile responsiveness fixes
- Animation polish

### Documentation

- Improve this README
- Add code comments
- Write tutorials
- Create video walkthroughs

## Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/ibrahim.git
cd ibrahim

# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Code Style

- Use TypeScript
- Follow existing patterns in the codebase
- Keep components focused and reusable
- Use Tailwind CSS for styling
- Maintain RTL support for Arabic

## Commit Messages

Write clear, concise commit messages:
- `fix: resolve mobile layout issue in sponsor grid`
- `feat: add Turkish language support`
- `docs: update installation instructions`
- `style: improve button hover states`

## Questions?

- Open an issue for technical questions
- Contact the maintainers for campaign-related questions

## Code of Conduct

Be respectful. This is a humanitarian project serving a real person in medical need. Treat all contributors and the beneficiary with dignity.
